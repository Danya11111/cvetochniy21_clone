# Локализация

## Принцип

Пользовательские, операторские и административные тексты поддерживаются на русском языке.

## Что локализовано

- Admin mini-SPA (навигация, кнопки, таблицы, уведомления).
- Операционные документы (rollout/runbook/checklists).
- Финальные pre-production отчёты.

## Что не переводится

- Имена файлов, переменных, таблиц, API routes, коды ошибок.
- Технические идентификаторы и контракты.

## Правило сопровождения

При добавлении новых UI/operator текстов:
1. Писать объясняющую часть на русском.
2. Не менять технические идентификаторы.
3. Избегать смешивания RU/EN в одной пользовательской фразе без необходимости.
